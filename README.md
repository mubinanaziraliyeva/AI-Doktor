# AI-Doktor

AI Doktor
AI Doktor — bu sun’iy intellekt yordamida tibbiyot xizmatlarini yanada samarali va qulay qilishga qaratilgan veb-ilova. Sayt foydalanuvchilarga tezkor aloqa, ma’lumot almashish va qo‘llab-quvvatlash imkoniyatlarini taqdim etadi.

✨ Asosiy imkoniyatlar
Kontakt formasi: Foydalanuvchilar ism, familiya, email va telefon orqali tezkor bog‘lanishlari mumkin.

Ijtimoiy login: LinkedIn, Facebook, Email va Instagram orqali kirish imkoniyati.

Responsive dizayn: Zamonaviy UI/UX tamoyillari asosida qurilgan, mobil va desktop qurilmalarda qulay ishlaydi.

Gradient fon va vizual elementlar: Violet va kulrang overlay bilan professional ko‘rinish.

🛠️ Texnologiyalar
HTML5 / CSS3 / JavaScript

TailwindCSS + DaisyUI (stil va komponentlar)

Responsive layout (grid va flex asosida)

📌 Maqsad
AI Doktor loyihasi tibbiyot sohasida sun’iy intellektdan foydalanib, foydalanuvchilarga tezkor va ishonchli xizmat ko‘rsatishni maqsad qiladi. Sayt orqali bemorlar va mutaxassislar o‘rtasida samarali aloqa o‘rnatiladi.
